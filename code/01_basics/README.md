# Block 1: Python Grundlagen

Dieser Block führt in Python ein. Die Beispiele laufen direkt auf deinem Computer ohne Mikrocontroller.

## Beispiele

| Datei             | Thema                                                      |
| ----------------- | ---------------------------------------------------------- |
| `01_variables.py` | Variablen und Datentypen                                   |
| `02_if.py`        | Bedingungen – simulierter Sensorwert                       |
| `03_for.py`       | For-Schleife – Winkel von 0 bis 180                        |
| `04_while.py`     | While-Schleife – der Grundrhythmus eingebetteter Programme |
| `05_sine.py`      | Sinuswelle – mathematisch generierte Bewegung              |

## Ausführen

In Thonny einfügen und ausführen.

## Aufgaben

1. Finde in `02_if.py` einen Wert für `distance`, der `"weit weg"` ausgibt.
2. Ergänze `02_if.py` um eine vierte Kategorie deiner Wahl.
3. Lass `03_for.py` nur Winkel ausgeben, die größer als 90 sind.
4. Lass `04_while.py` bei 10 aufhören, statt unendlich weiterzuzählen.
5. Ändere `05_sine.py` so, dass der Wert zwischen `0` und `100` liegt.
