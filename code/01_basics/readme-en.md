# Block 1: Python Basics

Introduction to Python — no microcontroller needed. All examples run on your computer.

## Examples

| File | Topic |
|---|---|
| `01_variables.py` | Variables and data types |
| `02_if.py` | Conditions — simulated sensor value |
| `03_for.py` | For loop — angles from 0 to 180 |
| `04_while.py` | While loop — the core pattern of embedded programs |
| `05_sine.py` | Sine wave — mathematically generated motion |

## Running

```bash
python 01_variables.py
```
