# Block 2: Outputs

First steps with the Pico — controlling LEDs and servos.

## Requirements

- Raspberry Pi Pico
- Copy `library.py` from the root `code/` folder to the Pico alongside the example file
- For servos: SG90 on pin 28 (and pin 27 for the second servo)

## Examples

| File                | Topic                               |
| ------------------- | ----------------------------------- |
| `01_led.py`         | Blink the onboard LED               |
| `02_blink.py`       | LED with non-blocking Blinky object |
| `03_servo.py`       | Toggle servo between 0° and 90°     |
| `04_servo-fade.py`  | Smoothly sweep servo from 0° to 90° |
| `05_multi-servo.py` | Control two servos simultaneously   |
