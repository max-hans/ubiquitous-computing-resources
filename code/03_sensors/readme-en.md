# Block 3: Sensors

Reading sensors and connecting them to outputs.

## Requirements

- Raspberry Pi Pico
- Copy `library.py` from the root `code/` folder to the Pico alongside the example file
- Light sensor (analog) on pin 28
- Push button on pin 15 (with pull-down)
- Servo on pin 27

## Examples

| File | Topic |
|---|---|
| `01_light-sensor.py` | Read light sensor (ADC) |
| `02_button.py` | Read button and print value |
| `03_button-led.py` | Button pressed → LED on |
| `04_light-servo.py` | Light value controls servo angle |
